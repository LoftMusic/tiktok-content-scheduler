### Goal
Build a machine learning-based MIDI generator that learns from a dataset of 85 ritornele MIDI files to generate new ritornele-style melodies.

#### 5. Pattern-Based Generator (CURRENT BEST)
- Extracts note patterns from dataset (pitch + duration transitions)
- Generates using pattern transitions
- Supports multiple keys (Amin, Emin, Dmin, Fmin, Cmaj) and BPMs (100, 120, 126, 85)
- Generates 128-note melodies (8 measures)
- **Status**: Latest run (20:09) produced 15 files - user said "se pare ca au sunat mai bine"
