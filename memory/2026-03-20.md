## splice-desktop project — critical fixes today

### Root cause: asar archive broke Python spawn
- Python's native `open()` syscall doesn't understand Electron's asar virtual filesystem
- `app.asar\src\record.py` was treated as a literal path, not an archive entry
- Symptom: "can't open file ... app.asar\src\record.py: No such file or directory"
- Fix: set `"asar": false` in package.json build config → src/ is now a real directory on disk
- Also fixed RECORD_SCRIPT path from `app.asar\src\` to `app\src\`

**Root cause fixes applied:**
1. **Bug: null addEventListener crash** — minimizeBtn/closeBtn were referenced but the custom titlebar HTML had been removed. This JS TypeError crashed the entire renderer script before init() could run. Fixed by removing those lines.
